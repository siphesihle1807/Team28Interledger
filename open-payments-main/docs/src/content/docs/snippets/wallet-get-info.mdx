For a client to request a grant from an authorization server, the client must first verify the validity of the wallet address and get the URL of the wallet's authorization server.

These code snippets enable an authenticated client to verify a wallet address, get the basic information required to construct a new transaction, and discover the auth serverâ€™s URL. Note that unauthenticated clients can also call the Wallet Address API to get wallet address information.
